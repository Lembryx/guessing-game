# Guessing Game 🎲

This is my **first programming project**! I've been studying programming for about **one month**, and after reading a book about programming, I got really interested in the world of coding. I decided to create this game to practice my skills and explore the **C language**, which I really enjoyed working with.  

## About the Game

The game is a **number guessing challenge**:

- The computer randomly selects a secret number.
- Your goal is to guess the number within a limited number of attempts.
- The game provides hints if your guess is too high or too low.
- There are different difficulty levels: Easy, Medium, Hard, and Extreme, each with varying number ranges and attempts.
- Points are scored based on how close your guesses are to the secret number.

This project helped me practice **basic programming concepts** like variables, loops, conditionals, user input, and working with random numbers.  

## How to Play

1- Run the game executable in the terminal:

./guessing.exe

2- Follow the instructions:

* Choose a difficulty level.
* Guess the number within the allowed attempts.
* Pay attention to the hints about how close your guess is.
* Try to score as many points as possible!
* You can play again as many times as you like to improve your score.
